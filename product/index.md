# OverLog Product Page

## What OverLog Does

OverLog helps you:

- Log workouts with structured exercise data
- Track progress over time
- Monitor personal records and training consistency
- Sync workout data with Apple HealthKit (with your permission)

## Core Features

- Workout logging and history
- Progress analytics
- Exercise library and custom workouts
- Optional cloud sync for backup and cross-device continuity
