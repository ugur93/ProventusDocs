## Full Description

OverLog: Workout Progress Tracker is built for one thing: helping you train consistently and improve over time with progressive overload.

No account creation. No long and unecessary onboarding prosess.
Create your program, open a workout, and start logging. Just pure workout and gym logging

### Why this app

- Simple to start, create your own program and start tracking
- Powerful program and exercise customization
- Clear progress feedback after every workout
- Live activity

### Core features

- Fast set logging for both strength and cardio
- Create custom and personalized programs with weeks, days, and exercises
- Advanced exercise settings (rep ranges, AMRAP, tags)
- Progress graphs across workouts
- Set tagging (dropsets, negatives, tempo training)
- Personal records (PRs): weight, reps, volume
- Previous set context + suggested next weight
- Plate calculator with configurable bar/plate inventory
- Live Activity and reminder notifications support during active workouts
- Optional Apple Health integration
- Built-in stopwatch and workout timer
- Drag-and-drop handles & pre-filled sets for effortless gym logging
- Complete data export with offline functionality
- Timeline view showing your complete fitness journey with any exercise

### Built for progressive overload

- Track what you actually lifted
- See previous performance while logging
- Get practical next-step guidance based on progression settings
- Review trends and PRs to adjust intelligently

### Free and Pro

Free includes core logging and tracking.
Pro unlocks advanced editing, deeper analytics, Apple Health integration, ICloud Backup & Sync and expanded customization.

#### Free

- Log unlitimed workouts, sets, reps, and weight
- Build and run workout programs
- View progress charts and workout history
- PR visibility and baseline tracking
- Local-first usage with no account required

#### Pro

- Advanced exercise settings (rep ranges, AMRAP, tags)
- Unlimited custom programs (Free is limited to max 1)
- Advanced tools (including in-workout plate calculator access)
- Extended progress windows (including 90-day and longer periods)
- Enhanced progression workflow and premium planning controls
- Apple Health integration
- ICloud sync & Backup

### Privacy and control

- Works without account creation
- Data stored locally with optional iCloud backup/sync
- No ads
